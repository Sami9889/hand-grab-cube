# src modules

This folder contains small modules used by the etcGrab Cube app. Files are intentionally minimal stubs to be expanded:

- `hud.js` — on-screen HUD helper
- `multiview.js` — simple fusion helpers for multi-camera world landmarks
- `calibration.js` — calibration workflow stub
- `controllers.js` — map VR controller poses to avatar joints
- `retarget-full.js` — placeholder for full retarget algorithms
- `haptics.js` — helper to pulse controllers
- `io.js` — save/load helpers
- `debug.js` — thin logging wrapper
- `tracker-multi.js` — multi-tracker manager (stub)
